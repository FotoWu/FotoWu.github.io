CSS

1. box-sizing属性值

content-box:在宽度和高度之外绘制元素的内边距和边框，宽度为content

border-box: 为元素指定的任何内边距和边框都在已设定的宽度和高度内进行绘制，宽度为content+padding+border

inherit: 继承父元素box-sizing属性的值

2. CSS选择器权重

a.important
b.内联，如style=””，权值为1000
c.ID选择器，如#content，权值为0100
d.类，伪类和属性选择器，如.content，权值为0010
e.标签和伪元素选择器，如div p，权值为0001
f.通配符、子选择器、相邻选择器等，如*、>、+，权值为0000
g.继承的样式没有权值

3.动画两个属性的区别和适用场景

transition: 需要借助伪类、js、@media触发，通过transitionEnd事件添加循环，用于实现简单的动画

animation: 自动触发，用animation-iteration-count属性定义循环次数；用于复杂的动画，可以定义关键帧，以此来控制每一帧的动画效果

JavaScript

1. 闭包

定义：函数对象可以通过作用域相互关联起来，函数体内部的变量都可以保存在函数作用域内

原理：每次调用函数时，都会为它创建一个新的对象用来保存局部变量，把这个对象添加至作用域链中。当函数返回时，从作用域链中将这个绑定变量的对象删除。如果不存在嵌套，也没有其他引用指向这个绑定对象，它就会被当做垃圾回收掉。如果定义了嵌套函数，每个嵌套的函数都拥有一个作用域链，并且这个作用域链指向一个变量绑定对象。但如果这些嵌套的函数对象在外部函数中保存下来，那么它们也会和所指向的变量绑定对象一样当做垃圾回收。如果这个函数定义了嵌套，并将它作为返回值返回或存储在某处的属性里，这时就会有一个外部引用指向这个嵌套的函数，就不会被当做垃圾回收，并且它指向的变量绑定对象也不会被当做垃圾回收。

注意：多个嵌套函数共享一个作用域链，多个闭包共享同样的私有变量或变量。注意不要在函数内循环创建闭包，因为所有的闭包都共享变量，返回的是同一个值。

2. 作用域链

定义：与一段JavaScript代码相关联的一个对象列表或链表

作用：定义了这段代码“作用域中”的变量

方法：从链中的第一个对象开始查找，如果找到了就直接使用这个属性的值，如果没找到就继续找，直到找不到为止，最后没找到抛出一个引用错误异常(ReferenceError)

函数不嵌套：作用域链上有两个对象：第一个是定义函数参数和局部变量的对象，第二个是全局对象。当调用一个函数时，创建一个新的对象来存储它的局部变量，并将这个对象添加至保存的那个作用域的链上，同时创建一个新的更长的表示函数调用作用域的链。

函数嵌套：作用域链上至少有三个对象，每次调用外部函数时，作用域链都是不同的。

3. 原型链

定义：每个对象都有一个prototype属性，称为原型，而原型的值也是一个对象，因此它也有自己的原型，这样串联成一条原型链，原型链的链头是object，它的prototype比较特殊，是null

作用：用于对象继承，比如新建一个数组，数组的方法便从数组的原型上继承而来

应用：当访问对象的一个属性时，首先查找对象本身，找到则返回；若未找到，则继续查找其原型对象的属性。若整个原型链未找到则返回undefined。

4. 作用域链和原型链的区别

作用域链：闭包相关，自顶向下找变量，没找到返回ReferenceError（引用错误）

原型链：继承相关，自底向上找原型（属性),没找到返回undefined
