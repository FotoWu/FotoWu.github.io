1.Variable

  1. What is weakly-typed?
  • ANS: declare without specific type and the availability of modifying type in assignment
  1. What is variable scope?
  • ANS: the defining block of this variable in source code
  • Including: global scope and functional scope(local)
  1. What is Hoisting?
  • ANS: Variables can be used before they are declared, all of the declared variables are taken on the top of the function
    (function is also hoisting)
  1. Attribute Variable

  • When declaring a global variable, we actually define an attribute for the global object “window”, and this variable cannot be deleted using “delete” if it has been declared using “var”.

      var xiaoqiang = 1;
  console.log(xiaoqiang,window.xiaoqiang);//1 1
  delete xiaoqiang;//false
  console.log(xiaoqiang,window.truevar);//1 1

  • In ECMAScript, the global variable is the attribute of global object, and the local variable is the attribute of some object called in the function. It is called declarative environment record in ECMAScript5. Also, JavaScript allows to use “this” to refer global object.

6.Class

  1. Class Creation

    a. ES5 and Earlier version

     // Animal Class

 // Attribute
 this.name = name || 'Animal';

 // Instance method
 this.sleep = function(){
 console.log(this.name + 'is sleeping!');
 }
 }

 // Prototype method

 Animal.prototype.eat = function(food) {
 console.log(this.name + 'is eating: ' + food);

    b. ES6

  2. Class Inheritance

a. ES5: Prototypal Inheriatance, Constructor Inheritance, Combined Inheritance, Parasitic Inheritance

  • Prototypal Inheriatance: prototype => new an empty object => Father class, can’t implement multiple inheritance

      function Cat(){ }
  Cat.prototype = new Animal();
  Cat.prototype.name = 'cat';

  • Constructor Inheritance: copy father instance to child, not use prototype, can implement multiple inheritance

      function Cat(name){
  Animal.call(this);
  this.name = name || 'Tom';
  }

  • Combined Inheritance: call father constructor twice

      function Cat(name){
  Animal.call(this);
  this.name = name || 'Tom';
  }
  Cat.prototype = new Animal();
  Cat.prototype.constructor = Cat;

  • Parasitic Inheritance: create a class with no instance method

      function Cat(name){
  Animal.call(this);
  this.name = name || 'Tom';
  }
  (function(){
      var Super = function(){};
      Super.prototype = Animal.prototype;
      Cat.prototype = new Super();
  })();

b. ES6: use of ‘extends’ keyword

class Point {
}

class ColorPoint extends Point {
    constructor(x, y, color) {
    super(x, y); // call constructor(x, y) from father
    this.color = color;
  }

  toString() {
    return this.color + ' ' + super.toString(); // call toString() from father
  }
}
}
