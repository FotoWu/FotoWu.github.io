3. Status Code

Status Code Meaning Details
200 OK Request successfully, used in GET and POST
301 Move Permanently Requested resource is moved to new URI permanently, return new URI for browser to automatically redirect
302 Found Requested resource is moved to new URI temporarily
304 Not Modified Requested resource is not modified, return no resource
400 Bad Request Request contains grammar error which server can’t understand
401 Unauthorized Requires user identity authentication
403 Forbidden Refuses the request
404 Not Found Server cannot find resource according to the clinet request
500 Internal Server Error Server internal error, can’t finish the request

4. What happened when inputing a URL to display the webpage?

a. Find Server IP
- in cache: Browser => System => Router
- in hosts
- in DNS

b. constructs a http request based on IP and port number

c. encapsulates http request in a TCP package

d. transfer the TCP package through Transport Layer, Network Layer, Datalink Layer and Physical Layer to server

e. server interpretes the package and sends out response(html) to the client

f. client constructs DOM tree and CSSOM tree based on the returned html

g. render JavaScript

5. Deep Clone

var deepCopy = function(obj) {
    if(typeof obj !== 'object')
        return;
    var newObj = obj instanceof Array ? [] : {};
    for(var key in obj) {
        if(obj.hasOwnProperty(key)) {
            newObj[key] = typeof obj[key] === 'object' ? deepCopy(obj[key]) : obj[key];
        }
    }
    return newObj;
}
