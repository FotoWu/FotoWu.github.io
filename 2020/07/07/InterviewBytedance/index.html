  1. 如何判断空对象
a. 最常见的思路，for...in... 遍历属性，为真则为“非空数组”；否则为“空数组”
for (var i in obj) { // 如果不为空，则会执行到这一步，返回true
    return true
}
return false // 如果为空,返回false

b. 通过 JSON 自带的 stringify() 方法来判断:
if (JSON.stringify(data) === '{}') {
    return false // 如果为空,返回false
}
return true // 如果不为空，则会执行到这一步，返回true

  1. 说一说强缓存和协商缓存

都是从cache里读

强缓存状态码是200，协商缓存状态码是304，强缓存直接从cache获取数据，协商缓存先与服务器协商后获取，与强缓存相关的字段：cache-control:expires，与协商缓存相关的字段：Last-Modified, If-Modified-Since, If-None-Match
